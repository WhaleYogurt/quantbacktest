# Changelog

## [0.2.0] - 2025-11-12

- Data layer hardening: `DataSettings`, provider retries/backoff, validation upgrades.
- Engine core overhaul: multi-segment scheduler, metadata checkpoints, metrics analyzer integration.
- Portfolio enhancements: multi-currency cash, margin reserves, trade logging, leverage summaries.
- Strategy framework: indicator helpers, registry, momentum + mean-reversion strategies, new examples.
- Metrics/reporting: CLI analyzer, markdown/JSON reports, richer docs.
- CI pipeline: GitHub Actions lint + test jobs mirroring `test.bat`.

## [0.1.0] - Initial scaffold

- Project skeleton, placeholder engine, initial tests and documentation.
